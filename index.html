Banking infrastructure,

Venn-Vault is a full-stack banking API written in pure Node.js — no Express, no shortcuts. Custom routing, raw HTTP, bcrypt, JWTs, and a PostgreSQL ledger under the hood.

Built to understand, not just ship.

Every layer of this project was written intentionally — to strip away the abstractions and understand what's actually happening inside a web server.

⚙

Zero-Framework Architecture

No Express. Routing, request parsing, and response handling are implemented directly on Node's native http module — every byte is accounted for.

🔒

Airtight Security Model

Passwords hashed with bcrypt, sessions managed via stateless JWTs, and secure HTTP-only cookies. Auth that would hold up in production.

🗄

Relational Database Layer

Persistent user accounts and a proper transaction ledger backed by PostgreSQL. Real schemas, real queries, no ORM magic.

💳

Core Banking Operations

Full deposit, withdrawal, and fund transfer flows — all routed through the custom parser and hitting the database on every transaction.

🛡

Admin Control Panel

A separate admin interface for viewing all accounts, freezing or deleting users, and inspecting any transaction history in the system.

🧪

Sandbox Economy

Deposits and withdrawals intentionally allow unlimited dummy inputs, so you can stress-test the routing and database layers without worrying about fake balances.
